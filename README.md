A Python web app template

## Resources provided

* Flask - http://flask.pocoo.org
* Flask-restful - http://flask-restful.readthedocs.org 
* Celery - 
* Swagger -

## Quick Start

*Installation*

Create a virtual environment using the requirements.txt pip file.

*Testing*

Run tests using pytest
```
(venv)> py.test
```
